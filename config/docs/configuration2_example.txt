## J1939 Protocol Details
J1939 is a higher-layer protocol based on CAN bus, commonly used in heavy-duty vehicles and industrial applications.

### PGN (Parameter Group Number)
The PGN identifies the message type in J1939 communication. This configuration monitors PGN F004.
